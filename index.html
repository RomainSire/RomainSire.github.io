Works

Social Network

Fullstack development of a social network for a fictive company. Angular is used for the frontend part, Node.js with the framework Express for the backend. Data is stored in a MySQL database. The main features are user's account, list an research of users, publication of text and/or pictures with resize on the go, like/dislike a publication, comments system, notifications system, moderation with an admin special account.

Check out the video

Piocheur de Bobines

Frontend development of a cinema application. I used Angular to develop the frontend of this app, and the public database "The Movie Database" and its API for the backend. The user can see what are the trending movies, sort them by genre, search for a specific movie, and of course display the detail of a specific movie and its trailer. Very interesting and fun app' to code, and I learned a lot about Unit Testing! :)

Check out the app

Crème Solaire ou Parapluie

Frontend development of a weather application. I wanted to learn how to use Vue.Js. It's a very simple app, but I still managed to plug a router, and a data store with Vuex (probably overkill, but I wanted to learn!). I used Google Places API to get the gps coordinates, and to update the background image, and then I used the OpenWeather API to get the weather data. NB: The app currently doesn't work because..... Piquante

Backend development of a sauces ranking application. This REST API is developped with Node.js and the Express Framework, and the data is stored in a MongoDB database. A user can add a new sauce, and like or dislike other user's sauces. A special attention is given to security to make it as safe as possible.

Oricam

Frontend development of a e-commerce website for a vintage camera company. The backend API was already coded with Node.js, I developped the frontend with Vanilla JavaScript (no framework). The main features are fetch all the products details from the backend and display it to the user, add one or several products to the cart, ask the user his contact details, and then give the order id to the client.

La chouette agence

SEO optimization of an already existing website. Keword were analysed to target the most efficient words. The modifications performed where: Natural SEO (title, meta, semantic tags, url), Performances to finally reach 99% on GTMetrix (images, external ressources loading, cache), Accessibility (contrast, aria labels, forms).

To-Do List

Fullstack development of a simple To-Do List with PHP. You can add a task, its description, its deadline, and the priority. You can then crosscheck a task to visualize it's done, and also delete all checked tasks. I used PHP with no framework to make it work. I wanted to learn how to manipulate a file with PHP so I stored the data in a file for testing, but for production I would advise to use MongoDB instead.

Check out the video

My Paint

I used html canvas and Vanilla JavaScript to code it (no framework) in a very object-oriented approach. Functionnalities are scarce: one brush, one ereaser, different colors, different sizes. But that's plenty to have fun!
